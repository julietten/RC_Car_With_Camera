RC Car with Camera Stream

To see the live stream:

  1. Check that both the Raspberry Pi and monitor are on the same wifi network
  2. Download VLC Media Player on both the monitor you are watching the stream on and the Raspberry Pi
  3. Download rpi.py onto Raspberry Pi
  4. Enable camera in the Raspberry Pi configurations settings
  5. Run rpi.py on Raspberry Pi
  6. Open tab Media -> Open Network Stream -> add url of Raspberry Pi (ex. http://[insert RPi IP address]:8160)
